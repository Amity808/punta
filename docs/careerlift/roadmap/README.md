# CareerLift — 6-Week Delivery Roadmap

This document outlines the phased development and delivery timeline for the **CareerLift MVP**.

---

## 6-Week Release Plan

The MVP scope includes all core features required to deliver a functional, gamified interview practice experience:

| Week | Focus | Core Deliverables |
| :--- | :--- | :--- |
| **Week 1** | [Foundation: Auth & Onboarding](./week-01.md) | User signup, registration validations, multi-step onboarding preference configuration. |
| **Week 2** | [Dashboard & Design System](./week-02.md) | Shared UI elements, responsive dashboard layout, welcome card, dynamic daily question panels. |
| **Week 3** | [Questions Module](./week-03.md) | Question grid library with filter headers, response editor (draft and submit), collapsible answer panels. |
| **Week 4** | [Company Interview Database](./week-04.md) | Searchable company directories, timeline stages visualization, contribution forms. |
| **Week 5** | [Streaks & Profile Settings](./week-05.md) | Timezone-aware streak tracking, historical calendar grid, profile page, notifications settings. |
| **Week 6** | [Polish, QA & Launch](./week-06.md) | Mobile responsiveness check, accessibility audits (visible focus, ARIA tags), deploy to Cloudflare & Fly.io. |

---
